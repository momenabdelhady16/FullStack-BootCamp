المهارات والمسار الدراسي

★ أساسيات وهيكلة لصفحة:
← الهيكل الرئيسي: كتابة وسوم الهيكل الثابتة.
← نوع المستند: تعريف المتصفح على اصدار html الحديث.
← ترميز اللغة: دعم الحروف العربية عن طريق UTF-8.
← وصف الموقع SEO: تهيئة الموقع لمحركات البحث.
← عنوان التبويب: تسمية الصفحة في شريط المتصفح.
← التعليقات: تنظيم الكود.
★ العناصر النصية وتنسيق المحتوى:
← العناوين: تدرج حجم وأهمية العناوين من الأكبر إلى الأصغر.
← الفقرات: كتابة النصوص الطويلة والمقالات.
← النزول لسطر جديد: كسر السطر برمجيا عن طريق < br >.
← الخطوط الأفقية: وضع خط فاصل بين الأقسام.
← أوسمة التنسيق:
  • جعل الخط عريض لبيان الأهمية.
  • جعل الخط مائلاً للتمييز.
  • تحديد النص كأنه بقلم تحديد.
★ الروابط والصور والقوائم:
← الروابط:
  • روابط خارجية تفتح في صفحة جديدة.
  • روابط داخلية للتنقل بين أقسام نفس الصفحة.
← الصور: إدراج صور وفهم المسارات مع تحديد الأبعاد والوصف البديل.
← القوائم:
  • قوائم غير مرتبة Unordered Lists.
  • قوائم مرتبة Ordered Lists.
  • قوائم التعريف المصطلحية Description Lists.
★ الجداول:
← بناء جدول: إنشاء جدول كامل مع تفعيل الحدود.
← صفوف وأعمدة الجدول:
  • إنشاء صفوف.
  • إنشاء خلايا العناوين البارزة.
  • إدخال البيانات العادية
  • دمج الخلايا.
★ التقسيم الحديث:
← مربعات التقسيم (البلوكات): فهم عنصر < div > لجمع العناصر في كتل منفصلة تحت بعضها.
← العناصر المضمنة: فهم عنصر < span > للتعديل على جزء أو كلمة معينة داخل السطر.
← العناصر الدلالية: تخطيط الصفحة بطريقة يفهمها المتصفح.
★ الرموز الخاصة والوسائط المتعددة:
← الرموز الخاصة: كتابة رموز لا يمكن كتابتها من لوحة المفاتيح العادية (مثل ©).
← مشغلات الميديا:
  • إدراج ملفات صوتية مع إظهار أزرار التحكم.
  • إدراج ملفات مرئية (فيديو) مع إظهار أزرار التحكم.
★ أساسيات الإستمارات:
← حاوية الإستمارة: إنشاء نماذج باستخدام < form >.
← حقول الإدخال الأساسية:
  • حقول إدخال نصوص عادية.
  • حقول إدخال كلمات مرور.
